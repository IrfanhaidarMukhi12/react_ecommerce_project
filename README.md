## Run Locally

Clone the project

```bash
  gh repo clone IrfanhaidarMukhi12/react_ecommerce_project
  
```

Go to the project directory

```bash
  cd React_E-Commerce
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm start
```
